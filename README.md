# 🌤 Weather App with Angular

A modern Weather Application built using Angular.  
The app allows users to search for a city and view real-time weather information.

## 🚀 Live Demo
🔗 https://TokaFawzy.github.io/weather-app-with-angular/

---

## 📌 Features

- 🔍 Search for any city
- 🌡 Display current temperature
- ☁️ Show weather condition
- 📱 Responsive design using Bootstrap
- ⚡ Built with Angular (Standalone / Angular CLI)

---

## 🛠 Technologies Used

- Angular
- TypeScript
- Bootstrap
- HTML5
- CSS3
